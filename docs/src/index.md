# CalculusWithJuliaSquared.jl

Documentation for [CalculusWithJuliaSquared.jl](https://github.com/FourMInfo/CalculusWithJuliaSquared.jl), a personal pure-Julia fork of [CalculusWithJulia.jl](https://github.com/jverzani/CalculusWithJulia.jl), a package to accompany the notes "Calculus with Julia" for using `Julia` for Calculus.

To suggest corrections to the notes themselves, please submit a pull request to [https://github.com/jverzani/CalculusWithJuliaNotes.jl/](https://github.com/jverzani/CalculusWithJuliaNotes.jl/). The [Quarto](https://quarto.org/) pages makes this easy, as they have an "Edit this page" link.
